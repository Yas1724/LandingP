## About

A minimal, dark-themed personal landing page built for speed and simplicity. Showcases who I am, what I build, and where to find me.

## Tech Stack

| Technology | Purpose |
|---|---|
| [Qwik](https://qwik.dev) | Frontend framework |
| [TypeScript](https://www.typescriptlang.org) | Type safety |
| [Tailwind CSS](https://tailwindcss.com) | Styling |
| [Vercel](https://vercel.com) | Deployment |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/Yas1724/LandingP

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to see it locally.

## Deployment

Deployed on Vercel. Every push to `master` triggers a new deployment automatically.

```bash
git add .
git commit -m "your message"
git push origin master
```
